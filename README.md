# Simple-Physics-Calculator

A lightweight Command Line Interface (CLI) tool designed to solve fundamental physics equations. This project demonstrates basic Python logic, user input handling, and mathematical operations.

## Features
- **Velocity Calculation:** Computes velocity based on distance and time ($v = d/t$).
- **Force Calculation:** Computes force based on mass and acceleration ($F = m \cdot a$).
- **Input Validation:** Includes error handling for division by zero and invalid menu choices.

## Technologies Used
- **Language:** Python 3.x

## How to Run
1. Ensure you have Python installed.
2. Clone this repository.
3. Run the script using:
   ```bash
   python Physics-Calculator.py
